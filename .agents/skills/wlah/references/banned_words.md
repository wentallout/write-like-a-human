# The AI Register: Banned Words and Phrases Blocklist

> **DO NOT USE THESE WORDS.** This reference exists to identify the most common algorithmic prose tropes ("synthetic puffery" and "significance traps"). If you find yourself drafting these words, you are falling into the AI register and must rewrite the sentence completely.

## The Absolute Blocklist (Zero Tolerance)

### Adjectives & Descriptors
- pivotal
- transformative
- innovative
- dynamic
- robust
- seamless
- unparalleled
- comprehensive
- nuanced
- ever-evolving

### Nouns & Verbs
- delve (e.g., "let's delve into")
- tapestry (e.g., "a rich tapestry of")
- testament (e.g., "stands as a testament to")
- underscore (e.g., "this underscores the importance")
- landscape (e.g., "the evolving landscape")
- realm (e.g., "in the realm of")
- beacon (e.g., "a beacon of hope")
- journey (e.g., "embark on a journey")
- symphony (e.g., "a symphony of")
- paradigm (e.g., "a paradigm shift")

## Connectives and Transitions (Banned)
Instead of these rigid robotic transitions, simply begin the next sentence or use a conversational transition ("Here's the thing", "That said", "But honestly").
- Furthermore
- Moreover
- Additionally
- Consequently
- Therefore (unless in strict mathematical logic)
- In conclusion / To summarize
- Overall
- It is worth noting / It should be noted
